个人信息

体重：70kg（154磅）
基础代谢量：154 * 11 = 1696cal
代谢因子50%
维持现状所需的cal：1696+1696*50% = 2544cal
瘦体重：35.5kg（78.1磅）
每日蛋白质需求量：78.1 * 1.14 = 89.034g
每日从蛋白质摄取的cal：89.034 * 4 =356
除去来自蛋白的cal：2544 - 356 = 2187

饮食计划

高蛋白阶段

计划所需cal

2187+500 = 2687cal

每日需要从碳水和脂肪中获得的cal都等于

2687/2 = 1344cal

每日所需碳水化合物

1344/4 = 336g

每日所需脂肪

1344/9 = 150g

高蛋白阶段饮食计划

蛋白

刚训练完：总蛋白*25% = 22.5g
其余五餐：每餐67.5/5 = 13.5g

碳水化合物

刚训练完：100g高GI值碳水化合物
其余五餐：每餐(336-100)/5 = 47.2g

脂肪

刚训练完：30g
其余五餐：每餐(150-30)/5 = 24g

蛋白缩减阶段

计划所需cal

2544-120 = 2424cal（相当于减少30g蛋白的热量）

每日cal需求

2424+500 = 2924cal

每日碳水化合物和脂肪

各自2924/2 = 1462cal

每日碳水化合物需求

1462/4 = 366g

每日脂肪需求

1462/9 = 163g

蛋白缩减阶段饮食计划

蛋白质

训练结束后立刻摄取每日所需的所有的30g蛋白质

碳水化合物

训练结束：100g高GI碳水化合物
其余五餐：每餐(366-100)/5 = 53.2g

脂肪

训练结束：30g
其余五餐：每餐(163-30)/5= 26.6g

训练计划

训练次序：肩 臂+腹 腿 背 胸+腹 HIIT或休息

一周计划

周一（肩）

空手热身 4组

坐姿哑铃提拉（后束）4组*15次

俯身侧平举（后束）4组*15次

站姿推举（前中束）4组*10次

侧平举（后束）4组*15次

阿诺德推举（前中束）4组*15次

坐姿推举（中束）4组*15次

哑铃前平举（前束）2组*15次

杠铃片前平举+旋转 2组*15次

周二

平板交替伸手抬腿 4组*15次

平板支撑 力竭组*8

俄罗斯转体 8组*40次

哑铃弯举 6组*10次

锤式弯举 6组*10次

俯身哑铃后平举 6组*15次

周三

坐姿腿平举

1片-1组 20次
2片-2组 15次
3片-2组 12次
4片-2组 12次
3片-3组控制组 15次
2片-3组控制组 20次

俯卧腿弯举

1片-1组 20次
2片-2组 15次
3片-2组 12次
4片-2组 12次
3片-3组控制组 15次
2片-3组控制组 20次

周四

杠铃划船 4组*15次（2组1片，2组2片）

俯卧撑 150个

平板支撑 力竭组*8

周五

俯卧撑*500

周六

俯卧撑不限

周日

休息
